**Automated S3 Bucket Cleanup**

**Prerequisites**
- AWS Account
- Python 3.12 Runtime or higher
- Existing S3 Bucket
- IAM Role for Lambda
- Boto3 (included with Lambda)

**Step 1 – Create an S3 Bucket**

  1. Open AWS Console
  2. Go to Amazon S3
  3. Click Create Bucket
